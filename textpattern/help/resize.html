Image Size

Two images are automatically created from the uploaded original which is always kept and from which future resizings are made:

  • A regular size image. The size of this will be automatically the same as the last image that was uploaded or resized. If the image is rectangular then the longest side will be the same as the longest side of the previously handled image. It is also possible to force each image to be the same height or width by checkmarking either the width or the height field.
  • A thumbnail image can also be cropped to a square (from the left, middle or right) or left whole.
